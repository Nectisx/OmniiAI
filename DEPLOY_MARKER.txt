Deploy marker v8 — Fix Gemini 2.0 systemInstruction format

Problème : Gemini API v1beta a changé le format de systemInstruction.
Erreur reçue : "Invalid value at 'system_instruction' (type.googleapis.com/google.ai.generativelanguage.v1beta.Content)"

Solution : passer systemInstruction comme un objet Content
{ role: 'system', parts: [{ text: '...' }] }
au lieu d'une simple string.
